# sopel-mangadex

A MangaDex plugin for Sopel IRC bots

## Installing

Releases are hosted on PyPI, so after installing Sopel, all you need is `pip`:

```shell
$ pip install sopel-mangadex
```

## Configuring

The easiest way to configure `sopel-mangadex` is via Sopel's
configuration wizard—simply run `sopel-plugins configure mangadex`
and enter the values for which it prompts you.
